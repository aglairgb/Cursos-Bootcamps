Dart era pra ser um transpilador/transcompilador
=> tradutor que recebe código fonte de um programa em uma linguagem e o traduz para outra linguagem.
Um exemplo: Typescript.
=> Mesmo intuito, dart era pra ser um transpilador de javascript.
A linguagem se baseia em: JavaScript, Java, C#.
-
Flutter Engine (Skia) -> responsável pela renderização dos componentes em tela.
-
Biblioteca Foundation -> responsável pela comunicação da interface do usuário com o sistema operacional.
-
Design-specific -> Widgets com implementações prontas para android (google material) e ios (cupertino).
-
Vantagens Flutter:
- salvar em tempo de execução.
- faz chamada direto para pegar dados do sistema operacional, pois não possui uma bridge (camada que fica entre ios/android e componentes/hardware) - react native usa utiliza esse intermediário .
