<<Conceitos base sobre sistemas operacionais>>
- Um sistema operacional é um software que atua como intermediário entre o usuário e o hardware do computador.
- São responsáveis por traduzir as entradas do usuário para a linguagem de máquina , formando uma interface entre o usuário e o hardware, como também traduzir para o usuário as saídas geradas pelo hardware.
Usuário -> aplicação -> sistema operacional -> hardware

* Funções:
- Grenciamento de processos
- Gerenciamento de memória
- Gerenciamento de dispositivos
- Gerenciamento de arquivos
- Segurança
- Detecção de erros
- Agendamento de tarefas

* Recursos do SO's
- Fornece uma plataforma para executar aplicativos
- Gerenciamento de memória e CPU
- Fornece abstração do sistema de arquivos
- Fornece suporte de rede
- Fornece recursos de segurança
- Fornece interface de usuário
- Fornece utilitários e serviços de sistema
- Suporte ao desenvolvimento de aplicativos

Para executar tarefas, os sistemas operacionais possuem dois componentes:
*Shell
 - Seja por meio de uma interface com linhas de comando ou gráfica, o shell realiza ações básicas do sistema operacional para administrar dados.
Funções:
- Solicitar entradas do usuário
- Interpretar a entrada para o sistema operacional
- Manipular a saída do sistema operacional

*Kernel
- Núcleo do sistema operacional, uma ponte entre hardware e software.  O kernel que gerencia os recursos do sistema e permite que os programas façam uso deles.
Funções do kernel:
- Gereciamento de entrada-saída
- Gerenciamento de memória
- Gerenciamento de processos para execução de aplicativos
- Gerenciamento de dispositivos
- Controle de chamadas do sistema

Exemplos de SO's:
- Microsof windows
- MacOS
- Linux
- Android

<<Windows>>
- O Windows é o principal sistema operacional da microsoft, o padrão de fat para computadores domésticos e empresariais.
- É um sistema opereracional baseado na interface gráfica do usuário (GUI), introduzido em 1985 e foi lançado em várias versões. 
- A principal linguagem de programação do código-fonte das várias versões do windows é o C e algumas partes com C++ e Assembly.
- Até a versão 3.11, rodava com 16 bits (com um update chamado win32s adicionando o suporte 32 bits).
- As versões a partir do XP e Server 2003 estão preparadas para tecnologia 64bits.  
- Devido sua grande popularidade, o windows é um grande alvo das ameaças cibernéticas.

<<SO's baseados em Linux>>
- O SO Unix implementado em 1969 por Ken Thompson, Dennis Ritchie, Douglas Mcllroy e Joe Ossanna
- Lançado pela primeira vez em 1971, era escrito em assembly
- Em 1973 foi reescrito na linguagem C por Denni Ritchie
- O Kernel do Linux foi, originalmente, escrito por Linus Torvalds do departamento de ciência da computação da universidade de Helsinki, Finlândia, com a ajuda de vários programadores através da Usenet.
- É um s.o open-source, ou seja, pode ser editado e distribuído sem o pagamento de royalties
