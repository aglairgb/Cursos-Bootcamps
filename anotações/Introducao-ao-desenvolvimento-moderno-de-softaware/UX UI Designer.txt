UX Desgin (User Experience)

- O design da experiência do usuário (Ou UX design) é o processo que visa melhorar a satisfação do usuário com um produto ou serviço, melhorando a usabilidade, a acessibilidade e até mesmo a satisfação proporcionada na interação.
- Envolve pesquisas. Quem será o usuário do softwarwe? Qual média de idade? O usuário pode ter dificuldade na interação com o produto? 
- O profissional, com suas pesquisas aponta em forma de desenho os melhores locais para posicionar os elementos.
- Ferramentas utilizadas no UX/UI design: Figma, Adobe XP, entre outras. Ótimos para desenhar um protótipo de onde ficarão cada elemento.
- Esse profissional faz a pesquisa, entende qual a pessoa que vai utilizar o aplicativo, então ele desenha protótipos para determinar onde ficará cada elemento da tela. Também necessário compreender o objetivo desta operação para poder trazer melhor conforto ao usuário.

UI Design (User Interface)

- O designer de interface do usuário é o profissional responsável em criar o que o usuário irá ver e utilizar o produto. Este profissional entende padrões visuais que podem ajudar o usuário na experiência de utilização do software.
- Profissional focado em cores, tipografia, microinterações e estilos.

Enquanto o UX designer determina as posições de botões, por exemplo, o profissional UI designer é o responsável por desenhá-lo, determinar fonte, cor e etc..

UI designer, ou designer de interface do usuário, promove a criação e o desenvolvimento da interface explorada pelo usuário em um produto ou serviço.
- tipografia, tipo de letra, escala de cores sempre pensando na persona/ no público atingido e suas necessidades.

UX Designer = comportamento (pesquisa, protótipos, personas, objetivos)
UI Designer = Aparência (Design, tipografia, cores, layouts)
